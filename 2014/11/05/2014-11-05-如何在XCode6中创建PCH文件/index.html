如何在XCode6中创建PCH文件

2014-11-05
aries58
aries58

pch文件主要用于分类文件头文件导入,方便全局使用。公共宏的定义,项目api管理等。
以下是PCH文件创建步骤:

1.新建文件 command + N

2.选择iOS/Mac -> Other -> PCH File -> 输入pch文件名,例如: YourProject-Prefix.pch

3.在Xcode6中,选择Project -> Build Settings, 然后搜索: Prefix Header

4.在Apple LLVM 6.0分组可以看到Prefix Header

5.输入YourProjectName/YourProject-Prefix.pch

6.Clean项目: command + shift + K

7.编译项目: command + B
