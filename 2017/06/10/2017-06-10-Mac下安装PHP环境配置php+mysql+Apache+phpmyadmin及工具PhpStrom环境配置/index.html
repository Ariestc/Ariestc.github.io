Mac下安装PHP环境配置php+mysql+Apache+phpmyadmin及工具PhpStrom环境配置

2017-06-10

Mac默认已安装Apache和PHP环境,只需开启即可

一、开启Apache
sudo apachectl start
附:
sudo apachectl stop（关闭）
sudo apachectl -k restart(重启)
开启后浏览器输入: http://localhost 回车会显示”It works!” 表示Apache开启成功

二、开启PHP
1.修改Apache配置文件
1> 打开终端输入命令: sudo vim /etc/apache2/httpd.conf
2> 找到#LoadModule php5_module libexec/apache2/libphp5.so删除本行前面”#”字符即可
3> mac下Apache默认文件夹路劲为: /Library/WebServer/Documents 在该目录下创建一个test.php文件，文件中输入如下内容: <?php phpInfo(); ?>
浏览器输入:http://localhost/test.php 开是否显示PHPinfo页面
4> 若不成功 打开终端，输入命令：
sudo vim /etc/apache2/httpd.conf
找到①DocumentRoot “/Library/WebServer/Documents”
②DocumentRoot “/Library/WebServer/Documents”

将这两出””中的路径修改为test.php存放的路径,删除前面的”#”;
5>浏览器输入localhost/test.php,出现PHP的info页,则表示成功。

三、安装MySQL
1> 官网下载: https://dev.mysql.com/downloads/mysql/
2> 安装，注意安装完后会弹出对话框，务必保存临时密码到文件夹，以备后用
3> 启动MySQL
系统偏好设置-> MySQL
4> 点击Start MySQL Server 启动
5> 打开终端，输入命令：
/usr/local/mysql/bin/mysqladmin -u root -p’填写之前保存的临时密码’ password ‘新设置的密码’

四、安装phpmyadmin
1> 下载地址: https://www.phpmyadmin.net
下载后更改文件夹名为: phpmyadmin 并放在/Library/WebServer/Documents/目录下
2> 终端进入 cd /etc/apache2
3> ls 查看文件（输入nl httpd.conf 可给此文件加上行号）
4> vim httpd.conf（修改此文件）此步骤待确定
5> 浏览器输入: localhost/phpMyAdmin 用户名填写: root 密码填写: 新设置的密码 登陆成功即可。

五、PhpStrom环境配置
1> Mac OS X 中没有默认的php.ini文件,有对应的模版文件php.ini.default
自行拷贝修改终端输入:
sudo cp /private/etc/php.ini.default /private/etc/php.ini
2> 在Apache目录下创建PHP路径
如: /Library/WebServer/Documents/myphp
myphp为Apache路径下自定义创建的文件夹，用于放置php文件
3> 修改端口号，phpstrom默认端口号为63342，而Apache端口号为80
Preferences->Buid,Execution
添加project设置
Name为:Apache
Type为In place
Web sever root URL:http://localhost
点击apply 然后点击ok
4>点击Mappings选项
设置localpath为: /Library/WebServer/Documents/
点击apply 然后点击ok
